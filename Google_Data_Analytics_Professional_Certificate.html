Google Data Analytics Professional Certificate

On this page you'll find all of the certificates I achieved as part of the Google Data Analytics Professional Certificate course, as well as some information about the course.

You can view the full course on the Coursera website by clicking the post title. A PDF version of the certificate is available to download by clicking the button below.

Google Data Analytics Program Content

At the time of completion, the Google Data Analytics Professional Certificate was split into 7 modules/courses and a capstone project.

I have broken down the 7 courses below:

  • Course 1 - Foundations: Data, Data, Everywhere
  • Course 2 - Ask Questions to Make Data-Driven Decisions
  • Course 3 - Prepare Data for Exploration
  • Course 4 - Process Data from Dirty to Clean
  • Course 5 - Analyze Data to Answer Questions
  • Course 6 - Share Data Through the Art of Visualization
  • Course 7 - Data Analysis with R Programming

For the full, in-depth course content, please click on each of the courses, which will take you direct to the course website on Coursera.

The GDAPC gave me the foundations of being a data analyst. I particularly liked that this course not only taught me new skills, but also how to think like a data analyst.

Some of the skills I learnt were, but not limited to:

  • Data Analysis Process
  • Data Visualisation using Tableau
  • SQL Programming in Google Big Query
  • Stakeholder communication
  • Data-driven Decision Making mindset
  • Data Cleaning
  • Introduction to R programming language
  • Data Integrity & Ethics

In total, the GDAPC took me about 2 months of determined studying everyday to complete. I would highly recommend this course for anyone with an interest in data analytics or businesses looking to up-skill their employees in data analytics.
